# Multiplayer Tic Tac Toe Game

A real-time multiplayer Tic Tac Toe game with player profiles, house building system, and live chat functionality.

## Features

- **Multiplayer Gameplay**: Play online with other players in real-time
- **30-Second Turn Limit**: Each player has 30 seconds to make their move
- **Player Profiles**: Build your house based on wins
- **House Building System**: Earn construction stages for each win
- **Real-Time Chat**: Chat with opponents during matches
- **Player Statistics**: Track wins, losses, and win rates

## Technology Stack

- **Backend**: Node.js, Express, Socket.io
- **Frontend**: React, Redux
- **Database**: MongoDB
- **Authentication**: JWT
- **Deployment**: Docker
